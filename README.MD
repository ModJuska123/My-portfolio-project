# Welcome to My-Portfolio-Project

Welcome to My-Portfolio-Project, where I show my web development journey through a collection of projects and skills. This repository reflects my passion for creating dynamic and visually appealing web applications using React and CSS. 

## Table of Contents
- Installation
- Run the app

## Installation

1. Clone this repository:
```
git clone https://github.com/ModJuska123/My-portfolio-project
```
2. Navigate into the project directory:

```
cd My-Portfolio-Project
```
3. npm install

```
npm install
```

## Run the app

1. Start the development server:
```
npm start
```
2. Open your browser and visit http://localhost:3000 to view the app.
